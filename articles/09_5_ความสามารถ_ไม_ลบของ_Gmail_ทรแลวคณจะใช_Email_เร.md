# 5 ความสามารถ (ไม่) ลับของ Gmail ที่รู้แล้วคุณจะใช้ Email เร็วขึ้น 3 เท่า !!! (อย่างน้อย)

Auto-Advance Emailเวลาเราอ่าน email เสร็จแล้วเราก็มักจะกด reply หรือ archive email นั้นใช่ไหมครับ โดยปกติ Gmail ก็จะไม่ได้ทำอะไรต่อแต่เราสามารถที่จะทำให้หลังจากที่เรากด reply หรือ archvie email นั้นๆ แล้ว ให้ Gmail แสดง Email ถัดไปขึ้นมาทันทีได้ทำให้เราไม่ต้องไปคอยกด email ใหม่ๆ ขึ้นมาอ่าน (ประหยัดเวลาไปอีกกกก)วิธีใช้งานง่ายครับ เข้าไปที่ Setting (1)>> See all settings (2) >> Advanced (3) >> Auto-Advance (4) >> Enable (5) (ตามรูปด้านล่าง) (อย่าลืมกด Save Changes ด้านล่าง)3. กำหนดเวลาให้สามารถกดย้อนกลับการส่ง email ได้ (Undo Send Time Period)เวลาเรากดส่ง email ไปแล้วเนี่ย มันจะมีให้เรากด undo หรือ ย้อนกลับการส่ง (ตามรูปด้านบน) นั้นได้ใช่ไหมครับ แต่เจ้าปุ่มนี้เนี่ยมันอยู่แปปเดียว ส่วนใหญ่จะกดย้อนกลับไม่ทันหรือกว่าเราจะนึกได้ว่าอยากย้อนกลับ เจ้าปุ่มนี้ก็จะหายไปแล้ว (เวรกรรม)แต่มันมีวิธีตั้งค่าเพื่อให้เจ้าปุ่ม Undo Send เนี่ย อยู่ได้นานถึงครึ่งนาทีเลยวิธีการคือ Setting (1) >> See all settings (2) >> Undo Send (3) >> เลือกเวลาได้เลย (4) ตามรูปด้านล่างครับ (อย่าลืมกด Save Changes ด้านล่าง)เพียงเท่านี้เราก็มีเวลาได้หายใจเพิ่มขึ้นหลังจากเรากดส่ง Email เพื่อตัดสินใจว่าจะกด undo send หรือไม่ครับ4. แจ้งเตือน Email ที่น่าตอบหรือน่าติดตาม (Nudges)เคยไหมครับส่ง Email ถามอะไรใครไปหรือมีคนอื่นถามอะไรเรามา แล้วเราลืมตอบ (ม่ายยยยย) นั่นแหละครับ ปัญหานี้ Gmail คิดทางแก้มาให้เราแล้ว เพียงแค่เราเปิด Nudges feature ของ Gmail จะทำให้มีการแจ้งเตือนให้เราเมื่อเรายังไม่ได้ตอบ หรือเรายังไม่ได้รับการตอบกลับ โดย Gmail จะเด้งมาเป็น Email บนสุดใน Inbox ของเราเลยสำหรับ Feature นี้แนะนำให้เปิดทิ้งไว้สักช่วงเพื่อให้ Gmail เรียนรู้ก่อนนะครับ จะทำให้แจ้งเตือนอัตโนมัติได้ฉลาดมากยิ่งขึ้นวิธตั้งค่า Setting (1) >> See all settings (2) >> General (3) >> Nudges (4) แล้วติ๊กถูกได้เลย (5) ตามรูปด้านล่างครับ (อย่าลืมกด Save Changes ด้านล่าง)5. Gmail Offline Modeอินเตอร์เน็ตบ้านเราบางทีก็ผีเข้าผีออกครับ วันดีคืนดีก็อาจจะล่มได้ ถ้าเน็ตล่ม แล้วเราจะใช้งาน Gmail อย่างไร มาลองดูกันครับหมายเหตุ วิธีนี้ทำให้กินที่ในเครื่องเพิ่มมากขึ้นนะครับ แต่ถ้าทำงานกับ Email เยอะ ๆ ก็ถือเป็นทางเลือกที่ดีครับเผื่อในวันที่อินเตอร์เน็ตไม่ดีหรือไปในที่ที่ไม่มีอินเตอร์เน็ตจริงๆวิธตั้งค่า Setting (1) >> See all settings (2) >> Offline (3) >> Enable offline mail (4) >> เลือกว่าจะเอา Email ย้อนหลังกี่วัน (5) >> เลือกว่าถ้าเรา logout Gmail เครื่องนี้จะให้เก็บ Email นี้เอาไว้ในเครื่องนี้หรือไม่ (6) เสร็จแล้วอย่าลืมกด Save Changes ด้านล่างเมื่อตั้งค่าเสร็จแล้ว เวลาต้องการใช้งานให้เราไปที่ กล่อง Search ด้านบน แล้ว พิมพ์ mail.google.com เพียงเท่านี้ก็จะไปยังที่เก็บ Email offline ของเราแล้วครับหวังว่าบทความนี้จะเป็นประโยชน์ไม่มากก็น้อยนะครับ แล้วเจอกันใหม่บทความต่อไปครับRef: https://www.youtube.com/watch?v=UMeOh2hp0Zk&list=WL&index=10
