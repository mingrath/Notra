เป็นเรื่องที่หลีกเลี่ยงไม่ได้ การทำงานร่วมกับคนอื่นจึงเป็นเรื่องที่สำคัญมาก ตลาด Collaborative tools ก็โตขึ้นมากเป็นเงาตามตัว ทางเลือกแต่ก่อนก็มีไม่มากครับ หนึ่งใน Software ที่เป็นหนึ่งในตัวเลือกที่ดีที่สุดก็คือ Notion ครับNotion คือหนึ่งในทางเลือกที่ดีที่สุด (ก่อนหน้านี้)ด้วยเหตุผลเรื่องการโตขึ้นของการใช้งาน Collaborative tools ทำให้ Notion เองเมื่อ 3–4 ปีที่ผ่านมา ได้รับความนิยมสูงขึ้นอย่างมากโดยเฉพาะปีนี้ที่หลายๆ คนหันมาสนใจ Notion กันมากขึ้น ใช้งานเป็นทั้ง Productivity application คือใช้สำหรับ จด และใช้ทำงานร่วมกันกับเพื่อนร่วมงาน เปรียบเสมือนกับคลังข้อมูล หรือ Dashboard ของบริษัทก็ยังเป็นได้ครับซึ่ง Notion ก็ทำหน้าที่ตรงนี้ได้ดีแต่ Notion ก็ยังมีจุดอ่อนอยู่มากไม่ว่าจะเป็นความช้าหรือความหน่วงของมัน การทำงานร่วมกันเป็นทีมก็ยังไม่ดีเท่าทีควร การแจ้งเตือนต่างๆ ก็ยังติดขัด (เตือนบ้างไม่เตือนบ้าง) และที่สำคัญคือ Mobile app เข้าขั้นแย่กันเลยทีเดียวแต่อย่างว่าครับมีก็ยังดีกว่าไม่มีครับ เพราะครั้นเราจะไปใช้ Google docs หรือ Microsoft word ในการทำสิ่งเหล่านี้ มันก็ไม่ได้ออกแบบมาให้ทำแบบนั้นได้ครับด้วยเหตุผลเหล่านี้ Notion จึงเป็นหนึ่งในทางเลือกที่ดีที่สุดสำหรับการทำงานร่วมกับคนอื่น หรือ เป็น หนึ่งใน Collaborative tools ที่ดีที่สุดMicrosoft เห็นโอกาสทาง Microsoft เองก็คงเห็นว่าตลาด Collaborative tools นั้นยังมีช่องว่างอยู่มาก จึงกระโดดเข้ามาศึกษา และปล่อย Software ตัวใหม่ออกมาชื่อ“Microsoft Loop”ผมขอเรียกสั้นๆ ว่า loop นะครับ ซึ่งทาง Microsoft เองเพิ่งปล่อย Loop ออกมาเมื่อกลางสัปดาห์ที่ผ่านมานี้นี่เอง ซึ่งเป็นที่ฮึอฮาในวงการ Productive apps มากๆถ้าใครได้ดูหน้าตาของมันจะเห็นว่า หน้าตามันละม้ายคล้ายกับ Notion เหมือนจับวางแต่ๆๆๆ เพิ่ม Feature บางอย่างเข้าไป ซึ่งเจ้า Feature พวกนี้นี่แหละ มันเป็นสิ่งที่มาเติมเต็ม ทำให้ Notion เอง ก็ตกที่นั่งลำบาก เพราะ Feature ที่ทาง Loop ให้มานั้นมาปิดจุดอ่อนของทาง Notion ได้เป็นอย่างดีเดี่ยววันนี้ผมจะเปรียบเทียบให้ฟังครับ ว่าระหว่าง Notion กับ Loop มีอะไรแตกต่างหรือเหมือนกันบ้าง ไปชมกันได้เลยยยยยNotion vs Loopหน้าตาโดยรวมNotionLoopมาเริ่มกันที่หน้าตากันก่อนเลยครับ จะเห็นว่า หน้าตาเหมือนกันแทบจะ 100% เลยครับ ไม่ว่าจะเป็นแถบด้านซ้าย หรือ หน้าตาแต่ละหน้าที่มี Cover กับ Emoji เหมือนกัน สามารถตั้งค่าให้แสดงผลเป็น Thumbnail ได้เหมือนกัน สรุปคือ เหมือนกันหมดทุกอย่างในส่วนของหน้าตาครับMenuMenu bar ของ NotionMenu bar ของ Loopในส่วนของ Menu bar นั้นก็จะเห็นว่าใกล้เคียงกันมากๆ อีกแล้ว (ของ Loop เองตอนนี้ยังไม่ได้ออกมาให้ได้ทดลองใช้จริงเลยยังบอกไม่ได้ว่าจะมีอะไรมากน้อยเพียงใด) แต่จะเห็นว่าทางฝั่ง Loop จะมี Feature จำพวก Collaborative มากกว่าเช่น Voting table หรือ Status tracker ซึ่งน่าติดตามสุดๆ ว่าจะทำอะไรได้ดังนั้นเราจะเห็นว่าโดยภาพรวมของทั้ง 2 apps นั้นหน้าตาใกล้เคียงกันสุด แต่สิ่งที่ Loop เพิ่มเข้ามาคือ Collaborative Feature เดี่ยวเราไปดูกันครับว่า Collaborative Feature อะไรที่ทาง Loop เพิ่มเข้ามาและเทพสุดๆ3 สิ่งที่ทำให้ Loop เหนือกว่า Notionทาง Microsoft เรียก Loop ว่าเป็น Collaborative Canvasซึ่งก็ตามชื่อเลย จุดเด่นของ Loop คือการทำงานร่วมกันได้อย่างไร้รอยต่อแบบสุด เรามาดู Feature เด่นๆ ที่ Loop ชูมาสำหรับการทำงาน Collaborative1. Live componentและนี่คือหนึ่งใน Feature ที่อาจจะทำให้ Loop อาจจะเอาชนะ Notion ได้ แต่ก่อน Microsoft เรียกสิ่งนี้ว่า Microsoft Fluid แต่ตอนนี้เอามารวมอยู่ใน Loop ไปเลยLive component คือ Database ที่เป็นส่วนหนึ่งของ Dashboard ที่เราต้องการแสดงผลแบบ real-time และเราสามารถเชื่อมส่วนนี้ไปยัง Email หรือ Office อื่นๆ ได้โดยง่าย2. Sidebar มีการบอกว่าแต่ละคนทำงานอะไรอยู่ และมี Activities logอย่างที่บอกไปด้านบนครับว่าหน้าตา Sidebar ทั้งของทั้ง Notion และ Loop นั้นเหมือนกันมาก แต่มีสิ่งที่แตกต่างกันคือLoop นั้นจะบอกด้วยว่าตอนนี้ใครกำลังทำอะไรอยู่ที่ Page หรือหน้าไหนอยู่ และยังมี Notification (ซึ่ง Notion แย่เรื่องนี้มาก) บอกด้วยว่ามีใคร Mention หาเราแม้ว่าอาจจะดูเล็กน้อยนะครับ แต่สำหรับการทำงานร่วมกันเป็นทีม ผมว่านี่เป็น Game-changer เลย เพราะมันทำให้การทำงานร่วมกันลื่นไหล และไม่มีการตกหล่นเรื่องการแจ้งเตือนเลยยิ่งไปกว่านั้นด้านล่างจะมีส่วนที่เราสามารถดู Activities log ได้ ซึ่งส่วนนี้สำคัญมากๆ ครับสำหรับการทำงานแบบ Remote เพราะเราต้องสามารถ Monitor และ Track ได้ว่าใครแก้อะไรยังไงไปบ้างถึงแม้ว่า Notion จะมี Logs ให้ดู แต่ทำได้ไม่ดีเลยครับ เพราะฉะนั้น อันนี้อาจจะเป็นหนึ่งในส่วนที่ Loop เอาชนะ Notion ได้สบายๆ..3.
