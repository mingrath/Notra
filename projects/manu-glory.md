# Manu Glory

> Manchester United fan dashboard with match data, standings, and Supabase backend

## Problem
Manchester United fans need a centralized place to track match data and standings.

## Solution
A dedicated dashboard providing real-time match information and league standings.

## Tech Stack
- **Key tech**: TypeScript, Supabase.

## Key Features
- Match data tracking
- League standings
- Backend integration

## Highlights
Built with a Supabase backend for reliable data management.
