# Draw.io MCP

> MCP server enabling LLMs to create and open diagrams in draw.io — supports XML, Mermaid, and CSV formats

## Problem
LLMs often struggle to create and manipulate visual diagrams directly. Developers need a way to bridge the gap between AI-generated content and visual diagramming tools.

## Solution
An MCP server that allows LLMs to interact with draw.io for creating and opening diagrams. It supports multiple formats like XML, Mermaid, and CSV for versatile diagramming.

## Tech Stack
- **JavaScript**: Core language for implementing the MCP server and draw.io integration.

## Key Features
- AI-driven diagram creation
- Support for XML, Mermaid, and CSV
- Seamless draw.io integration

## Highlights
A powerful tool for visual communication. It enables LLMs to generate and manipulate diagrams, enhancing the visual capabilities of AI systems.
