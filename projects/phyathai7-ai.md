# Phyathai 7 AI

> AI Clinical Assistant — SOAP Generator + CEO Dashboard + LINE Chatbot

## Problem
Clinical documentation and management in veterinary settings can be time-consuming and fragmented. Veterinarians need efficient ways to generate SOAP notes and monitor clinic performance.

## Solution
A comprehensive AI assistant that automates SOAP note generation and provides a centralized dashboard for clinic management. It includes a LINE chatbot for easy interaction and accessibility.

## Tech Stack
- **TypeScript**: Robust language for building the assistant and dashboard.

## Key Features
- Automated SOAP note generation
- CEO performance dashboard
- Integrated LINE chatbot

## Highlights
A multi-faceted tool designed to streamline veterinary clinical workflows. It combines AI-driven documentation with management insights.
