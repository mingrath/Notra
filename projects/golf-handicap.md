# Golf Handicap

> Mobile-first PWA for pairwise handicap golf scoring with configurable handicaps, turbo holes, and game history

## Problem
Golfers need a mobile-friendly way to score pairwise games with complex handicap rules.

## Solution
A mobile-first PWA that handles configurable handicaps and game history.

## Tech Stack
- **Key tech**: TypeScript.

## Key Features
- Pairwise scoring
- Configurable handicaps
- Turbo holes
- Game history

## Highlights
Designed as a PWA for seamless mobile use on the course.
