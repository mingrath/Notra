# Claude Code Statusline

> Custom statusline for Claude Code with real-time rate limit usage percentages and progress bars

## Problem
Developers using Claude Code need a clear way to monitor their rate limit usage and progress during long-running tasks. Without real-time feedback, it's easy to hit limits unexpectedly.

## Solution
A custom statusline for Claude Code that provides real-time visibility into rate limit usage and task progress. It uses progress bars and percentages to give developers immediate feedback.

## Tech Stack
- **TypeScript**: Core language for building the statusline and handling real-time data.

## Key Features
- Real-time rate limit monitoring
- Visual progress bars
- Seamless integration with Claude Code

## Highlights
A developer utility for enhancing the Claude Code experience. It provides essential feedback for managing API usage and task progress.
