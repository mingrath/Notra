# Claude Code Notify

> Get notified on Mac, iPhone, and Apple Watch when Claude Code needs your input. Uses terminal-notifier + ntfy.sh + Claude Code hooks.

## Problem
Developers using Claude Code for long-running tasks may not be at their computer when input is required. This can lead to delays and inefficiencies in the development process.

## Solution
A notification system for Claude Code that alerts developers on their Mac, iPhone, and Apple Watch when input is needed. It uses terminal-notifier, ntfy.sh, and Claude Code hooks for seamless alerts.

## Tech Stack
- **Shell**: Core language for implementing the notification scripts and hooks.

## Key Features
- Multi-device notifications
- Integration with terminal-notifier and ntfy.sh
- Automated alerts for Claude Code input

## Highlights
A developer utility for enhancing the Claude Code workflow. It ensures that developers are always informed when their attention is required.
